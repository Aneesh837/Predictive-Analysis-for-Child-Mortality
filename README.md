# Predictive Analytics for Child Mortality
### Leveraging Machine Learning for Early Intervention

## Overview
This project develops a machine learning-based system to predict fetal health risk levels using cardiotocographic data. The research was published in the International Journal of Information Technology & Computer Engineering (IJITCE), Volume 13, Issue 1, 2025.

## Live Demo
https://predictive-analysis-for-child-mortality-mzqgfrcukzxxbc4d24umhu.streamlit.app/

## Dataset
- Size: 2,126 records x 22 features
- Target: fetal_health — 3 classes (Normal, Suspected, Pathological)

## Models Used
- Random Forest — 94.6% accuracy (MCC: 0.8474)
- Support Vector Machine (SVM) — 90.4% accuracy
- Logistic Regression — 87.8% accuracy

## Tech Stack
Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn, Plotly, Streamlit
